#5 Usability analysis

The current test users were able to complete all the tasks with no apparent difficulty as a result of the site improvements.
At the same time, participants will spend more time in the practical stage of organizing the reward racing, but it is easier for them to find the racing information(t1,2,6). After the number of function buttons is increased, users can find the target page more easily and spend more time filling in and submitting information(t3,4,5).
According to the comments and suggestions of users, the future website needs to make the following improvement:
1) Some hyperlinked buttons need to be made more visible
2) Express payment amount
3) Continue to improve the functions: including the display of competition data, record submission and display, as well as home page display, my release in personal page.
In addition, the AB test is still ongoing.
